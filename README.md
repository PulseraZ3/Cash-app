# Aplicación de Gastos Mensuales 
## Descripción
Esta aplicación, desarrollada en **Swift** y probada en el simulador de **iPhone 11**, permite a los usuarios llevar un control de sus gastos mensuales. Los usuarios pueden agregar, visualizar y categorizar sus gastos de manera sencilla, obteniendo una visión clara de sus finanzas personales.

---

## Funcionalidades

- Agregar gastos con detalles como:
  - **Nombre del gasto**
  - **Categoría** (Ej: Alimentación, Transporte, Entretenimiento, etc.)
  - **Monto**
  - **Fecha**
- Visualizar lista de gastos mensuales.
- Ver un resumen del total gastado por mes.
- Ver un grafico de gastos
- Interfaz amigable y fácil de usar adaptada para iPhone 11.

---

## Tecnologías

- **Lenguaje:** Swift  
- **Entorno de desarrollo:** Xcode  
- **Simulador utilizado:** iPhone 11  
- **Persistencia de datos:** CoreData (según implementación)
